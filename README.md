# Raccoon Ball / Industrial Sound Anomaly Detection

Repository for preprocessing, training, and experiment metadata for the industrial sound anomaly detection track.

## Includes
- preprocessing notebooks and scripts
- training notebooks
- manifest/config metadata
- model cards and training history
- method documentation

## Excludes
- raw audio data
- generated training corpus
- model weight binaries

## Notes
Generated datasets and large artifacts are intentionally kept outside Git.
